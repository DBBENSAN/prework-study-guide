Prework Study Guide

HTML

  • To begin an HTML doc, declare it using "!DOCTYPE html" between gt/lt symbols.
  • Elements refers to the entire element, from opening tag to closing tag.
  • <head> contains the metadata of the website.
  • <meta> elements contain information about the page.
  • Users won't see this information in either <head> or <meta> elements
  • Elements that are nested inside other elements are called child elements. The container is the parent element.
  • HTML headings are titles or subtitles that are meant to be displayed. Ranges from H1 -> H6
  • other common attributes: <hr>= horizontal rule, <p>= paragraph, <br>= line break

CSS

  • Declarations contain two important components: the property + value
  • The margin property indicates how much space we want outside of an element
  • The padding propert adds space around the content inside of an elemnent
  • * -> applies a rule to all visible elements on the page
  • To link an external css file, make sure to link after the last meta tag and before the title element
  • . + ___ to designate class selectors. HTML with this class will have these CSS properties

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • We use the add -A command to commit all files in the working branch.
  • git commit -m to associate a message with our commit. Make it short.
